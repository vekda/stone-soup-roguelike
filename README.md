# 地牢炖汤（Stone Soup 风格传统肉鸽）

一个浏览器里的传统 roguelike：随机房间地牢、回合制移动、撞怪攻击、拾取/使用物品、职业技能、升级、永久死亡。灵感来自《Dungeon Crawl Stone Soup》的传统肉鸽体验，但角色、敌人、道具和图片素材均为本项目原创。

## 运行

```bash
cd C:\Users\vekda\workspace\stone-soup-roguelike
npm install
npm run start
```

浏览器打开 Vite 显示的地址（通常是 http://localhost:5173）。

## 操作

- 方向键 / WASD / HJKL：移动或攻击
- Q：使用职业技能
- 数字 1-8：使用背包物品（瞬间动作，不触发怪物回合）
- 空格 / `.`：等待一回合
- 走向自己的召唤物：与召唤物交换位置，避免窄路卡死
- R：返回选角色界面

## 职业与技能

| 职业 | 定位 | 技能 |
|---|---|---|
| 铁锅战士 | 高防稳定 | 锅盖猛击：重击相邻敌人，并获得 2 回合减伤。 |
| 香料法师 | 脆皮爆发 | 辣雾爆燃：灼烧距离 5 内所有敌人。 |
| 菜刀游侠 | 机动刺杀 | 影步背刺：闪到近处敌人旁并发动强力背刺。 |
| 汤药医师 | 续航辅助 | 急煮药汤：回复生命；满血时获得临时防御。 |
| 酵母召唤师 | 召唤控场 | 发酵面团：在身边召唤一个会吸引敌人的面团伙伴；面团会攻击相邻敌人，伤害等于召唤师当前攻击。 |

## 内容

- 5 个可选职业：铁锅战士、香料法师、菜刀游侠、汤药医师、酵母召唤师
- 9 种敌人：地牢鼠、盐沼哥布林、胀气酵母、熏肉兽人、汤巫师、汤锅巨魔、锅炉傀儡、盐沼蟹王、石汤古龙
- 10 种物品：红汤药水、硬面包、铁菜刀、锅盖甲、火焰卷轴、辣椒油瓶、冰镇酸梅汤、发酵粉、净盐袋、浓汤盅
- 胀气酵母死亡会爆裂，汤巫师会治疗附近受伤敌人，盐沼蟹王是第一个小 Boss
- 所有角色、敌人、物品、召唤物、地形都有 SVG 像素风图片：`public/assets/`
